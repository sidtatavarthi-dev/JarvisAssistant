# JarvisAssistant

A voice-controlled AI assistant for Windows that opens websites, controls system settings, and manages browser tabs using voice commands.

## Features
- Voice-activated commands (say "Jarvis" to activate)
- Open popular websites (YouTube, Gmail, Discord, Netflix, etc.)
- Control system volume and screen brightness
- Close browser tabs by number
- Text-to-speech feedback

## Technologies
- Python
- speech_recognition (Google Speech Recognition)
- pyttsx3 (Text-to-Speech)
- pyautogui (System control)
- screen_brightness_control
